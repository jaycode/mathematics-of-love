new CurrentDetail()

ViewModel that relates to detail page, where single lifetimes are simulated.
This ViewModel also relates to current detail window displayed in app.mainViz.
This is initialized in app.data.currentDetail.
Singleton instance is available as app.vm.CurrentDetail.
