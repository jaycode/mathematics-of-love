addGoal()

A function to be bound to "Add Goal" button in the HTML. The first call to this function will show input text for user to enter the setting of this new goal, while the second call will confirm its creation.
Validates goal adding with function app.simulationAnalysis.validateGoal.

A function to be bound to "Cancel Add Goal" button in HTML. Pressing this will cancel the current goal adding process.
